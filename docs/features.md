# Features

## Product Management

- Add products
- Edit product details
- Delete products
- Manage product images
- Set price and stock quantity
- Filter products by category

## Category Management

- Add categories
- Edit categories
- Delete categories
- Assign products to categories

## Inventory Management

- Track stock changes
- Low stock indicator
- Inventory logs

## Order Management

- View orders
- Update order status
- Track payment status
- View order items

## Coupon Management

- Create coupons
- Set discount type
- Set expiry date
- Activate or deactivate coupons

## Dashboard

- Total products
- Total orders
- Total revenue
- Pending orders
- Low stock products

## Future Enhancements

- Customer frontend
- Online payment integration
- Product reviews
- Invoice generation
- Export reports
