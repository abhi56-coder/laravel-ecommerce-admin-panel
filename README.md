# Laravel E-commerce Admin Panel

A Laravel-based admin dashboard for managing products, categories, inventory, orders, users, coupons, and reports.

## Project Goal

This project demonstrates Laravel backend and admin panel development skills using practical e-commerce workflows.

## Tech Stack

- Laravel 10/11
- PHP 8+
- MySQL
- Blade / Bootstrap
- Git + GitHub

## Planned Features

- Admin authentication
- Product management
- Category management
- Inventory tracking
- Order management
- User management
- Coupon management
- Payment status tracking
- Sales dashboard
- Search and filters

## Main Modules

1. Authentication
2. Dashboard
3. Products
4. Categories
5. Orders
6. Users
7. Coupons
8. Inventory
9. Reports

## Database Tables

- users
- categories
- products
- orders
- order_items
- coupons
- inventory_logs

## Resume Line

Built a Laravel-based E-commerce Admin Panel with product/category CRUD, order management, inventory planning, coupon workflow, MySQL schema design, and dashboard documentation.
